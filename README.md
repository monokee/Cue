### Reactive module-driven state
```javascript
Cue.State('AppData', Module => ({
  ...
}));
```
### Declarative data-driven views
```javascript
Cue.UI('MainView', Component => ({
  ...
}));
```
### Composable application fragments
```javascript
const MyCue = Cue({
  state: 'AppData',
  ui: 'MainView'
});
```
